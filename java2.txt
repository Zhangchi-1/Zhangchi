#第19课
1.Java应用程序的执行入口是main( )的方法，它有固定的书写格式：
public static void main(String[] args) {…}
2.Java语言区分大小写
3.Java由一条条语句构成，每个语句以“;”结束
4.大括号成对出现
5.Java源文件以.java为扩展名，源文件的基本组成部分是class，如Hello类
6.一个源文件最多只有一个public类，其他类个数不限。编译后，每一个类都对应一个.class类
7.如果源文件包括一个public类，则文件名必须以public类命名
8.也可以将main方法写在非public类中，然后指定运行非public类

#第21课  Java的转义字符
1.在控制台输入Tab键，可以实现命令补全
1)\t  :  一个制表位，实现对齐的功能
2)\n :  换行符
3)\\  :  一个\
4)\"  :  一个"  (中文“”不用转义，英文“”需要转义)
5)\'  :  一个'
6)\r  :  一个回车  

课后作业：
public class Book {
	public static void main(String[] args) {
		System.out.println("书名\t作者\t价格\t销量\n三国\t罗贯中\t120\t1000");
		}
}

输出结果：
D:\javacode>javac Book.java

D:\javacode>java Book
书名    作者    价格    销量
三国    罗贯中  120     1000

#第22课  常见错误
1.找不到文件 ：1）文件名输入错误；2）文件不在当前目录下
2.应在xx文件中声明：主类名和文件名不一致，声明为public的主类应与文件名一致，否则编译失败
3.语法错误：根据报错去改错，例如：1，l ; 0 o ; 中英文符号

#第23课 注释
1.单行注释：//单行注释
2.多行注释： /*  注释... */
3.文档注释：注释内容可以被Jdk提供的工具javadoc解析，生成一套以网页文件形式体现的该程序的说明文档
                     /**
                        *
                        */
保存后，命令框输入 javadoc -d 保存路径 -author -version 文件名.java

#第26课  Java代码规范
1.选中代码，Tab键整体右移，Shift+Tab键，整体左移
2.md d:\\<file> 在D盘创建文件夹；rm d:\\<file>在D盘删除文件夹

#第28课  绝对路径和相对路径
1.相对路径：从当前目录开始定位，形成的一个路径
2.绝对路径：从顶级目录d开始，形成的路径
3.  ..\表示上一级目录

#DOS命令
1.查看当前目录的内容 ：dir，在当前目录下查看其他目录，如：dir d:\\program
2.切换到其他盘： cd 如：切换到C盘，cd /D c:
3.切换到当前盘的其他目录：cd + 绝对路径 OR 相对路径
4. 切换到根目录：cd \
5.切换到上一级：cd ..
6.查看指定目录下的所有子级目录 : tree + 路径
7.清屏 : cls
8.退出DOS ：exit
